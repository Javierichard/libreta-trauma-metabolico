# Libreta Trauma Metabólico - Reconstrucción Metabólica

Libreta digital de 16 semanas para el programa de Reconstrucción Metabólica del Dr. Javier Richard.

## 📦 Contenido del Proyecto

Este repositorio contiene:
- ✅ **Aplicación web completa** (React + TypeScript + Tailwind CSS)
- ✅ **59 páginas** listas para imprimir
- ✅ **Documentación completa** para recrear en Figma Design
- ✅ **Assets originales** (logos y foto)

## 🚀 Cómo usar este proyecto

### Opción 1: Ver la aplicación web

```bash
# Instalar dependencias
pnpm install

# Ejecutar en desarrollo
pnpm run dev
```

Abre tu navegador y navega por las 59 páginas de la libreta.

### Opción 2: Recrear en Figma Design (recomendado para imprenta)

Para exportar a PDF profesional para A Impresores:

1. **Lee la documentación:**
   - `DOCUMENTACION_COMPLETA.txt` - Guía completa en texto plano
   - `GUIA_FIGMA_PASO_A_PASO.md` - 17 pasos detallados
   - `ESPECIFICACIONES_FIGMA.md` - Especificaciones técnicas

2. **Importa los assets:**
   - `src/imports/Horizontal_Letras_Minimalista-1.jpg` - Logo horizontal
   - `src/imports/TMart_transarente.png` - Logo icono
   - `src/imports/Terno_-_Normal_-_Close_up.jpg` - Foto Dr. Richard

3. **Sigue la guía paso a paso** para recrear el diseño en Figma Design

4. **Exporta a PDF:**
   - `Libreta_TM_TAPAS_AIMPRESORES.pdf` (2 páginas)
   - `Libreta_TM_INTERIOR_AIMPRESORES.pdf` (57 páginas)

## 📐 Especificaciones de Impresión

- **Dimensiones:** 158 x 220 mm (A5 + 5mm sangrado)
- **Resolución:** 300 DPI
- **Páginas totales:** 59
- **Formato de entrega:** 2 PDFs (TAPAS + INTERIOR)
- **Imprenta:** A Impresores

## 📄 Estructura de la Libreta

1. **Portada** - Con logos y foto del Dr. Richard
2. **Introducción** - "Antes de empezar"
3. **Las Reglas** - 7 principios fundamentales
4. **Cinturones** - Sistema de progresión gamificado
5. **Mis Metas** - Página de objetivos personales
6. **Semanas 1-16** - Cada semana tiene 3 páginas:
   - Registro Semanal (tabla de hábitos diarios)
   - Evaluación Diaria (7 días con reflexiones)
   - Reflexión Semanal (métricas + cita de Tolkien)
7. **Notas** - 4 páginas para anotaciones libres
8. **Contraportada** - Página de transformación
9. **Contratapa** - Minimalista con logo
